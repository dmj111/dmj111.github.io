I don't blog very often for several reasons.  My technical excuse was
that blogging from emacs to one of the hosted services was too
painful.  This is an experiment to see if I can get around it.


`Org2Blog
<https://github.com/punchagan/org2blog>`_ allows you to post
from an OrgMode_ file in emacs, which is
pretty nice.  The only prerequisites are org mode and
xmlrpc.el_.  The setup is pretty
straightforward, as described in the ``README.org`` file.

.. _OrgMode: http://orgmode.org
.. _xmlrpc.el: https://launchpad.net/xml-rpc-el

Speaking of org and github, github appears to render ``README.org``
files in the main directory when you are viewing a repository.  That
is pretty awesome.

Code and latex snippets show up from org2blog too!  I am fine with the
default formatting, but I imagine this stuff could be tweaked to look
however I wanted.
