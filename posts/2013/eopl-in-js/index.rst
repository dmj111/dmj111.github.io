.. title: EOPL in Javascript

I have been working through EOPL_ again, this time programming in
JavaScript.  My JS experience is limited, but I finally know enough
that it isn't as painful anymore.

Some things that have helped:

- The debugging tools built into browsers
- Node.js
- Emacs -- `js2-mode`_, ``flymake-jslint``, and `PEG.js`_
- And finally knowing enough that I don't have to look stuff up all of
  the time.

.. _`PEG.js`: http://pegjs.majda.cz/
.. _EOPL: http://www.amazon.com/Essentials-Programming-Languages-Daniel-Friedman/dp/0262062798
